Github远程推送文件

📅 2026-03-18 📁 技术网络

1.在本地初始化 Git 仓库

打开终端（命令行），进入你的项目文件夹：

bash
cd /path/to/your/project
git init

2. 添加远程仓库地址

复制 GitHub 上仓库的 URL（HTTPS 或 SSH）。例如：

bash
git remote add origin https://github.com/你的用户名/仓库名.git

（如果是 SSH，使用 git@github.com:你的用户名/仓库名.git）

3. 添加文件到暂存区

将当前目录下的所有文件加入 Git 跟踪：

bash
git add .

（也可以只添加特定文件，如 git add README.md）

4. 提交文件到本地仓库

bash
git commit -m "first commit"

5. 推送至 GitHub

首次推送需要设置上游分支：

bash
git branch -M main          # 将本地分支命名为 main（与 GitHub 默认一致）
git push -u origin main     # 推送到远程仓库的 main 分支

推送成功后，刷新 GitHub 仓库页面即可看到你的文件。以后每次修改后，只需执行：

bash
git add .
git commit -m "更新说明"
git push
